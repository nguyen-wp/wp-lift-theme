# Run App

## Gulp 

Run `npm start` for dev

Run `npm run prod` for production

## SASS/JS Compiler 

This theme auto compiler SASS, CSS, JS to:

- `/dist/css/style.css` <= by SASS
- `/dist/css/theme.css` <= by CSS
- `/dist/js/theme.js` <= by JS
